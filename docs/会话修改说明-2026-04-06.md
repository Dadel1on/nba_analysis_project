## 4. 问题与修复记录

### 4.1 索引创建报错 1170
现象：
BLOB/TEXT column created_at used in key specification without a key length

原因：
prediction_snapshot.created_at 在目标库中为 TEXT/BLOB 类型，联合索引缺少前缀长度。

处理：
在 01_performance_indexes.sql 中做列类型判断：
1. TEXT/BLOB 时使用 created_at(32)
2. 其他类型时使用 created_at

### 4.2 API 返回 conference 为空
现象：
/api/teams 中 conference 为 null。

处理：
执行 03_dimension_enrichment.sql，补齐 conference 和 division。

结果：
/api/teams 已返回完整 East/West。

### 4.3 API 返回 position 为空或 UNK
现象：
部分球员 position 为空，后续规范为 UNK。

处理：
1. 03_dimension_enrichment.sql 统一规范位置值
2. NbaDataService 增加 UNK 展示映射

结果：
接口不再返回 null；UNK 可展示为“未知 (UNK)”。
